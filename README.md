# Vitor Teodoro - Personal Portfolio 🚀

A personal portfolio website built to showcase my journey in Computer Science, my technical stack, and my professional goals. 

Check out the live site here: [vitxt.github.io/VitorTeodoro](https://vitxt.github.io/VitorTeodoro)

## 🎯 Features
* **About Me:** Insights into my background and current studies at UEM.
* **Tech Stack:** Visual representation of the languages and frameworks I use.
* **Goals & Projects:** An overview of my current machine learning research and development goals.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
